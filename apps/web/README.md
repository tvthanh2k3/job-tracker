# Job Tracker - Frontend (Web)

Đây là ứng dụng Web (Frontend) của hệ thống Job Tracker, được xây dựng bằng **React + TypeScript + Vite**.

## 🚀 Hướng dẫy chạy ứng dụng

Để chạy phần Frontend này ở môi trường local, bạn cần mở terminal tại thư mục này (`apps/web`) và chạy:

```bash
# Cài đặt thư viện (nếu chưa cài)
npm install

# Khởi chạy server development
npm run dev
```

## 📚 Tài liệu chung
Để biết cách set up toàn bộ dự án (cả Backend và Frontend chạy cùng nhau, cấu trúc Database, v.v.), vui lòng tham khảo file **[README.md ở thư mục gốc](../../README.md)**.
